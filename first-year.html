First Year Timeline

Semester 1 (Aug - Dec)

• Master C Programming fundamentals
• Learn basic mathematics for CS
• Start Python programming basics
• Build simple console applications
• Join coding clubs and communities

Winter Break (Dec - Jan)

• Complete 1 free certification course
• Build 2-3 small projects
• Learn version control with Git
• Explore basic data science concepts

Semester 2 (Jan - May)

• Learn OOP concepts with C++
• Study software engineering principles
• Practice data structures basics
• Start mathematics for AI/ML
• Build intermediate projects

Summer Break (May - Jul)

• Complete advanced Python course
• Build portfolio website
• Contribute to open source
• Prepare for second year
• Start DSA practice
